# WordPress.org visual assets

This document defines the visual direction for the WordPress.org plugin assets.

## Main positioning

**Fast YouTube embeds. Nothing loads until click.**

The visual identity should focus on speed, lightness and avoiding unnecessary script loading before user interaction.

Privacy is part of the product, but the main visual message should be performance-first: pages stay light, YouTube is loaded only on demand.

## Assets to create

### Banner

Required size:

```text
1544 × 500 px
```

Suggested WordPress.org filenames:

```text
assets/banner-1544x500.png
assets/banner-772x250.png
```

Main copy:

```text
Privacy Lite Video Embeds for YouTube
Fast YouTube embeds. Nothing loads until click.
```

Visual concept:

- clean website/page background;
- central lightweight video placeholder;
- clear play button;
- small abstract “script blocks” or loading elements paused/disabled before click;
- visual sense of speed/lightness, such as subtle motion lines or a feather-like lightweight cue;
- no official YouTube logo;
- no trademark-heavy YouTube styling;
- no exaggerated legal/compliance claims.

Possible supporting micro-copy, if space allows:

```text
Local thumbnails · No unnecessary scripts · Load on click
```

### Icon

Required sizes:

```text
256 × 256 px
128 × 128 px
```

Suggested WordPress.org filenames:

```text
assets/icon-256x256.png
assets/icon-128x128.png
```

Icon concept:

- generic play button inside a lightweight frame;
- small speed/lightness cue;
- optional tiny shield or lock, but not dominant;
- avoid using the official YouTube logo.

## Style direction

- Modern, clean, lightweight.
- High contrast and readable at small sizes.
- Performance-first, privacy-aware.
- Minimal palette.
- Avoid dark cookie-banner/legal aesthetic.
- Avoid making the plugin look like a YouTube product.

## Preferred text hierarchy for banner

1. Plugin name: `Privacy Lite Video Embeds for YouTube`
2. Main promise: `Fast YouTube embeds. Nothing loads until click.`
3. Optional support line: `Local thumbnails · No unnecessary scripts · Load on click`

## Notes

The visual should communicate that embedded videos do not weigh down the page before interaction.

Think: a video embed that is “asleep” or “inactive” until the visitor clicks it.
